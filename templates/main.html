CorDet!

An online tool for detection of COVID-19 from chest xray radiographs

Get Started

How does it work

This web app uses a deep learning CNN model to classify the CXR images. More precisely MobileNet v1 pretrained on imagenet trained by transfer learning on a dataset of COVID-19 Positive and Negative images.

The Dataset

The dataset used here consisted of around 125 COVID-19 positive and around the same number of COVID-19 negative CXRs. COVID-19 positive images came from dataset published on this Github repo maintained by University of Montreal. COVID-19 negative CXRs came from a kaggle challenge
There is clearly lesser datat than what you would expect but with time better quality datasets will be available!

Important Note

This proof of concept is only for refernce purposes. Always refer a medical professional before taking any action!

Performance on test set

The dataset was randomly split into training, validation and test sets with the ratio 8:1:1. The model performed with a ttest set accuracy of more than 80%.
